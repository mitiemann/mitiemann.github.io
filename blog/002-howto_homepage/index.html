  1. Prerequisites

  • I assume you have basic understanding of git. This means you know how to clone, pull, commit, and push, and you know what branches are, how to create them and how to switch between them.

  • I assume you have a Github account.

  • I assume you have a basic understanding of programming and the Julia programming language. You have a recent julia version installed (at time of writing 1.10).
